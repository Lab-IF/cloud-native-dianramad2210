### Latihan 3 — Tantangan: Bandingkan Routing

React dan Node.js memiliki perbedaan utama pada fungsinya dalam pengembangan web. React digunakan untuk membangun tampilan antarmuka (frontend) yang berjalan di browser, sehingga fokus pada pengalaman pengguna dan komponen UI. Sementara itu, Node.js digunakan di sisi backend untuk menjalankan JavaScript di server, mengelola logika aplikasi, API, dan komunikasi dengan database. Singkatnya, React berperan dalam tampilan, sedangkan Node.js menangani proses dan data di belakang layar.
